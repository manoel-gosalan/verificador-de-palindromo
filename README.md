# 🔍 Verificador de Palíndromo

Aplicação web que verifica se uma palavra ou frase é um palíndromo, aplicando normalização de texto e validação de entrada.

## 🚀 Funcionalidades

- ✅ Validação para permitir apenas letras
- ✅ Normalização de texto (lowercase, remoção de acentos e espaços)
- ✅ Verificação de palíndromo
- ✅ Feedback visual dinâmico (sucesso/erro)
- ✅ Suporte à tecla Enter
- ✅ Layout responsivo estilo Japan Tech

## 🧠 Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla JS)
